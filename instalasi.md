# Skrip Instalasi

## BackEnd

1. Import Database
2. Install Dependency Codeigniter
    ```
    composer install
    ```
   ```
    composer update
    ```
3. Jalankan Server Untuk phpMyAdmin
4. Jalankan Server Codeigniter
    ```
    php spark serve
    ```

## Front End

1. Pastikan Sudah menginstall node.JS
2. Install Dependency
    ```
    npm install
    ```
3. Jalankan Server
    ```
    npm run dev
    ```
4. Build to production
    ```
    npm run build
    ```
